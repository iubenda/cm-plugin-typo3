Users Manual
============

The configuration of this plugin is very simple and easy to use.

1. First get your CMP-ID. Here is the link to register https://www.consentmanager.net/. We will provide you CMP-ID after registration.

2. Enter your CMP-ID in the Consent Manager Settings using the backend module (Click the Get started button below to access the general information)

3. Choose the blocking-mode (Automatic / Semi-Automatic)

4. Enter Custom HTML code(optional)

5. Click Submit button
